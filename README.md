# CheatSheet Template

## Introduction

This is a simple, responsive CheatSheet template designed for quick reference while coding. It contains basic HTML structures, code snippets, and examples with syntax highlighting. This template is easy to customize and expand according to your needs.

## Project Structure

- **index.html**: The main HTML file that contains the structure of the cheat sheet.
- **style.css**: The CSS file responsible for styling the cheat sheet, including layout, colors, and fonts.
- **prism.css & prism.js**: Files from Prism.js used for syntax highlighting of code snippets.
- **copyPNG.png**: A placeholder logo image used in the navigation bar.

## Features

- **Syntax Highlighting**: Code snippets are displayed with syntax highlighting using Prism.js, making it easy to read and understand the code.
- **Responsive Design**: The layout adjusts smoothly across different devices, including mobile phones, tablets, and desktops.
- **Custom Fonts**: Uses the "Potta One" font from Google Fonts, providing a distinctive and modern appearance.
- **Simple Navigation**: A basic navigation bar with a logo and a centered title for easy identification.

## How to Use

1. **Download or Clone the Repository**:
   - Download the ZIP file or use the following command to clone the repository:
     ```bash
     git clone https://github.com/Deepti-mayee/CheatSheet-Template.git
     ```

2. **Open the Project**:
   - Navigate to the project folder and open `index.html` in your web browser.

3. **Customization**:
   - **Add/Modify Code Snippets**: Edit the `index.html` file to add or change code examples as needed.
   - **Update Styles**: Modify `style.css` to change the appearance, such as colors, fonts, or layout.
   - **Change the Logo**: Replace `copyPNG.png` with your own logo by updating the `src` attribute in the `<img>` tag.

## Dependencies

- **Prism.js**: A lightweight and extensible syntax highlighter. Used here to highlight code snippets.
- **Google Fonts**: The "Potta One" font is used for the text to give it a unique style.

## Getting Started

1. **Ensure you have a text editor** like VS Code, Sublime Text, or any other to edit the HTML and CSS files.
2. **No installation is required** for viewing the cheat sheet; simply open the `index.html` file in any browser.
